# SavorGrid - Comprehensive Lifestyle Blog

SavorGrid is a modern, minimalist lifestyle blog website featuring six comprehensive categories: Fashion & Accessories, Health & Beauty, Home & Garden, Travel & Accommodation, Finance & Insurance, and Food & Beverage.

## Features

### 🎨 Design & User Experience
- **Minimalist Design**: Clean, modern interface with plenty of white space
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Animations**: CSS transitions and scroll-based animations
- **Intuitive Navigation**: Easy-to-use menu system with dropdown categories

### 📱 Pages & Functionality
- **Homepage**: Featured articles, category overview, and search functionality
- **Category Pages**: Filtered content by category with sorting options
- **Article Pages**: Detailed blog posts with rich content and images
- **Product Review Pages**: Comprehensive product evaluations with ratings
- **About Page**: Information about the blog and team
- **Contact Page**: Contact form with validation and FAQ section

### 🔍 Interactive Features
- **Search Functionality**: Real-time search across all content
- **Pagination**: Organized content navigation
- **Category Filtering**: Easy content discovery by topic
- **Mobile Menu**: Responsive navigation for mobile devices
- **Contact Form**: Validated contact form with user feedback

### 📝 Content
- **5 Featured Articles**: High-quality blog posts covering various topics
- **Product Reviews**: Detailed product evaluations with pros/cons
- **Rich Media**: High-quality images from Unsplash
- **SEO Optimized**: Proper meta tags and semantic HTML structure

## File Structure

```

├── js/

│   └── product.js         # Product page functionality
└── README.md              # This file
```

## Customization

### Colors
The color scheme is defined in CSS custom properties in `css/style.css`:
- Primary: #2c3e50 (dark blue-gray)
- Secondary: #ecf0f1 (light gray)
- Accent: #3498db (blue)
- Text: Various shades of gray

### Content
- Articles data is stored in `js/script.js`
- Product data is in `js/product.js`
- Easy to modify or extend with additional content
