# SavorGrid 使用说明

## 快速启动

1. **打开网站**
   - 双击 `index.html` 文件
   - 或者在浏览器中打开 `index.html`

2. **浏览功能**
   - 点击导航菜单中的 "Categories" 查看所有分类
   - 使用搜索框搜索文章
   - 点击文章卡片阅读完整内容

## 主要页面说明

### 首页 (index.html)
- 展示特色文章
- 六个分类的概览
- 搜索功能
- 分页导航

### 分类页面 (category.html)
- 按分类筛选文章
- 排序功能（最新、标题、热门）
- 文章数量统计

### 文章页面 (article.html)
- 详细的文章内容
- 产品推荐卡片
- 相关文章推荐
- 面包屑导航

### 产品页面 (product.html)
- 产品图片画廊
- 详细规格说明
- 优缺点分析
- 评分和评价

### 关于页面 (about.html)
- 网站介绍
- 团队信息
- 分类说明
- 统计数据

### 联系页面 (contact.html)
- 联系表单
- 表单验证
- 常见问题
- 社交媒体链接

## 特色功能

### 响应式设计
- 自动适配手机、平板、电脑屏幕
- 移动端优化的导航菜单

### 搜索功能
- 实时搜索文章标题和内容
- 按分类筛选结果

### 动画效果
- 平滑的页面过渡
- 鼠标悬停效果
- 滚动动画

### 用户体验
- 快速加载
- 直观的导航
- 清晰的视觉层次

## 技术特点

- 纯HTML/CSS/JavaScript，无需服务器
- 使用现代Web标准
- 优化的图片加载
- SEO友好的结构

## 浏览器兼容性

推荐使用以下浏览器：
- Chrome (最佳体验)
- Firefox
- Safari
- Edge

## 内容说明

网站包含以下内容分类：

1. **服装与配饰** - 时尚趋势和搭配指南
2. **健康与美容** - 健康生活和美容产品
3. **家居与园艺** - 室内设计和园艺技巧
4. **旅游与住宿** - 旅行指南和酒店推荐
5. **金融与保险** - 投资理财和保险建议
6. **食品与饮料** - 美食评价和餐厅推荐

每个分类都有精心制作的文章和产品评测，所有图片均来自高质量的Unsplash图库。

## 自定义修改

如需修改内容：
- 文章数据在 `js/script.js` 中
- 产品数据在 `js/product.js` 中
- 样式设置在 `css/style.css` 中
- 颜色主题可通过CSS变量修改
